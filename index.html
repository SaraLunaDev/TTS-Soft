Tutorial de Voces

Para usar las voces tendras que escribir el nombre de la voz con dos puntos entre parentesis.

(Martin:) Hola soy Martín.

Si quieres puedes usar varias voces en el mismo texto.

(Martin:) Hola soy Martín y ahora (Aerista:) soy Aerista.

Tutorial de Sonidos

Para usar los sonidos tendras que escribir el nombre del sonido entre parentesis.

Estoy muy triste (llorar) voy a beber agua (servir-agua).

Voces y Sonidos

Puedes combinar el uso de voces con el uso de audios.

(Arthur:) Que rica esta cerveza.. (tragar) (Aerista:) Deja de beber borracho! (puñetazo) ya vale eh!?

Avanzado

Al clicar un boton se te copiara el id en vez de la voz, de esta forma puedes si quieres ahorrarte caracteres

(3:) Hola soy Arthur, (Arthur:) y yo tambien lo soy. Toma (sartenazo) y toma tambien (25)
